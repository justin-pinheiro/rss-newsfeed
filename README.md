# RSS to PDF

## Description

Ce projet est un simple workflow pour l'extraction d'articles de journaux/magazines.

### Contexte

J'ai mis en place ce projet car je souhaitais pouvoir facilement avoir accès à des articles interessants, et pouvoir les lire sur ma tablette SuperNote. 

### Features

- Récupération des articles via des [flux rss](https://fr.wikipedia.org/wiki/RSS) ciblés. 
- Compilation des articles dans un fichier pdf.
- Sauvegarde des fichiers pdf dans un google drive.
